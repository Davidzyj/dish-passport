# 使用文档

## App 简介

Dish Passport（味游中国 / 中国味めぐり）是一款面向 iPhone 的离线中国菜单与文化指南。用户可以搜索菜名、查看点餐提示、理解菜品文化，并把想尝试或喜欢的菜保存在设备本地。

## 主要功能

### 探索

打开 App 后默认进入“探索”页。可以在搜索框输入英文、拼音或中文，例如 `mapo`、`xiaolongbao`、`麻婆`。搜索结果来自本地内置内容，不需要网络。

点击菜品进入详情页，可查看：

- 英文/中文/拼音/日文菜名
- 辣度
- 地区/菜系
- 口味关键词
- 主要食材
- 点餐表达
- 文化背景

### 保存与反馈

菜品详情页提供三个操作：

- 收藏：加入全部收藏
- 想尝试：加入想尝试清单，同时自动收藏
- 喜欢：加入喜欢清单，同时自动收藏

操作后会显示短反馈。返回探索、菜系或收藏页后，列表会立即显示最新状态。数据会保存在设备本地。

### 菜系

“菜系”页展示川菜、粤菜、江南、北方、湘菜、点心、面食、小吃等分类。进入菜系详情后可以查看地区风味说明、点餐建议和代表菜。

点击“全部加入想尝试”会把该菜系代表菜保存到想尝试清单。

### 收藏

“收藏”页按本地状态分区展示：

- 想尝试
- 已喜欢
- 全部收藏
- 最近浏览

如果没有保存内容，会显示空状态提示。

### 点餐短语

探索页和短语页提供常用点餐表达。进入短语详情后可复制中文短语，最近使用会保存在本地。

### 设置

设置页提供：

- 语言选择：自动、English、简体中文、日本語
- 隐私政策入口
- 支持入口
- 重置收藏数据
- 版本号

设置页不会显示 Bundle ID。

## 语言逻辑

如果用户显式选择语言，App 按用户选择显示界面。若用户选择“自动”或从未选择，App 根据系统语言/地区推断：

- 中文系统或中国大陆/港澳台地区：简体中文
- 日语系统或日本地区：日语
- 其他情况：英文

## 隐私说明

App 本身不需要账号，不连接后端，不包含广告或分析 SDK。收藏、喜欢、想尝试、最近浏览和语言选择只保存在 iPhone 本地。

隐私政策和支持网页只有用户点击设置页中的链接时才会打开。
