中国味めぐり

プライバシーポリシー

施行日：2026年6月13日

中国味めぐりはローカル優先の iPhone アプリです。アカウントは不要で、広告、分析 SDK、ユーザー追跡はありません。

データ収集

このアプリは個人データを収集しません。保存した料理、お気に入り、食べたいリスト、最近見た項目、言語設定は端末内に保存されます。

ネットワークアクセス

アプリがサーバーへ能動的にデータを送信することはありません。プライバシーポリシーとサポートのリンクは、ユーザーがタップした場合のみブラウザで開きます。

第三者

データを第三者と共有せず、第三者による追跡も使用しません。
